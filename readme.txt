Description:
You can get random countries by inputting a number, the number should be between 5-10. Then you will have list of random countries with related information.

Technologies:
Stack: TS, SASS. Bundler: Parcel. Pattern: MVC. 

Quick Setup:
Download the repo by using in your terminal git clone https://github.com/Murad-Git/didactic-waffle then go to the main folder and start the project by:
cd didactic-waffle

Install all dependencies:
npm i
or
yarn install

Run in development mode:
npm run start
or
yarn start

To run in production mode:
npm run build
or
yarn build

Bundled files for production will be in distr folder
